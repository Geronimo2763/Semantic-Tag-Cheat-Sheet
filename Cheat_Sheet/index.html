Semantic Tags Cheat Sheet

Name Tag Function
Header <header></header> A semantic tag that usually represent a container for indroductory content or a set of navigation links.
Navigation <nav></nav> A semantic tag that usually contains a set of navigation links.
Section <section></section> A semantic tag that groups content, typically with a heading.
Main <main></main> A semantic tag that represents the main content of the page.
Footer <footer></footer> A semantic tag that contains information about the author of the section, copyright data or links to related documents.
Article <article></article> A semantic tag specifies independent, self-contained content - it should make sense on its own.
Aside <aside></aside> A semantic tag that represents a portion of a document whose content is only indirectly related to the document's main content.
